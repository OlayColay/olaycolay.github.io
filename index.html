Tournament of Tamers

As a developer for Buh! Gaming, I work on the first-ever dragon vs. dragon MOBA. My accomplishments so far include implementing AI opponents that use 3D A* pathfinding, refactoring the game to be fully compatible with controllers, and using Google Analytics to record hundreds of different statistics before compiling them into a Data Studio document.

Skreech!

"Skreech!" is a project that I led a team in creating during the first half of 2021, as a part of ACM Studio at UCLA. Featuring two styles of platforming, the game revolves around being able to attach ropes and tether beams to walls and objects.

Try it out!

In Balance

A project for the Ludum Dare 49 game jam, "In Balance" is an RPG where I had to use all of my knowledge in Unity to lead a team of programmers, artists, and writers in completing within 72 hours. Unique to this game, I created timing mechanics for attacking and deefending by using C# coroutines and callbacks for Unity's Input System.

Try it out!
